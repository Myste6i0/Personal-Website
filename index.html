My work

IoT

The Automated Plant Cultivation System is an IoT prototype designed to optimize plant care by automatically maintaining soil moisture and light intensity. Using an Arduino module, it ensures plants receive the right amount of water and light. Real-time data is transmitted via an ESP32 WiFi module and can be monitored through a PHP website, allowing users to track conditions remotely without controlling the system.

Learn more

PHP Web App

Online FoodHub is a robust PHP web application designed to streamline the food ordering process. Users can browse a comprehensive menu from a single restaurant, select their desired food items, and place orders effortlessly. The platform ensures that each order is processed efficiently and delivered promptly to the user's specified location, providing a convenient and reliable dining experience from the comfort of their home.

Machine learning

The Hotel Booking Cancellation Prediction project uses data science and machine learning to forecast cancellations and improve revenue management. Analyzing key features from a detailed dataset, the project trains various models, with the Random Forest Classifier being highly accurate. It includes a Flask-based web interface, offering stakeholders actionable insights and an easy tool for managing cancellations.
